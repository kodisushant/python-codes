''' 1.Student Result Analysis System
Write a python program where A class teacher wants to prepare a result summary report. The system should calculate total marks, count students, find highest and lowest marks, and calculate average.

Sample Input

4
75
82
90
60

Sample Output

Total Marks: 307
Number of Students: 4
Highest Mark: 90
Lowest Mark: 60
Average: 76.75
'''

marks = []
for i in range(n):
    marks.append(int(input()))
# print(marks)
total = sum(marks)
max_marks = max(marks)
min_marks = min(marks)
avg=total/n

print("Total Marks:",total)
print("Number of Students:",n)
print("Highest Mark:",max_marks)
print("Lowest Mark:",min_marks)
print("Average:",avg)

''' Write a Python program where a supermarket wants to generate a daily bill summary.

The program should:
Store item prices in a list.
Display total number of items.
Calculate total bill amount.
Find the cheapest item price.
Find the costliest item price.

Note: Do not use built-in functions like sum(), min(), max() use for loop to Calculate total, cheapest and costliest


Sample Input

5
120
80
200
150
50

Sample Output

Total Items: 5
Total Bill: 600
Cheapest Item: 50
Costliest Item: 200
'''
n=int(input())
price =[]
for i in range(n):
    price.append(int(input()))
maxx=price[0]
minn=price[0]
summ=0
for j in range(n):
    if price[j] > maxx:
        maxx=price[j]
    if price[j]<minn:
        minn=price[j]
    summ+=price[j]
print("Total Items:",n)
print("Total Bill:",summ)
print("Cheapest Item:",minn)
print("Costliest Item:",maxx)
